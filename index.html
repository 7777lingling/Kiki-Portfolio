Technical Challenges

解決過的問題

這些經驗來自實際部署、資料搬遷與維運問題，重點不只是做出功能，也包含定位問題與修正系統設定。

GitHub Actions 部署失敗

解法
建立 Self-hosted Runner，讓部署流程能連接內部環境與 IIS。
學到的內容
CI/CD 不只要會寫 YAML，也要理解執行環境、權限與部署目標。

IIS 404 / 500

解法
檢查站台路徑、應用程式集區、Web.config 與 .NET Hosting Bundle。
學到的內容
後端部署錯誤需要從伺服器設定、執行階段與應用程式紀錄一起排查。

Azure SQL 搬遷

解法
確認 Schema、資料同步、連線字串與雲端防火牆規則。
學到的內容
資料庫搬遷需要同時考慮結構、資料一致性與應用程式連線設定。

TLS 憑證更新

解法
處理 CSR、SSL 憑證申請與 IIS / Tomcat 匯入設定。
學到的內容
HTTPS 上線牽涉憑證鏈、伺服器綁定與到期維護，不能只看程式碼。
